What is GNOME?

GNOME makes the look & feel of the GNU/Linux Operating System
very Familiar. Self described as: An easy & elegant way to use your computer
GNOME gives a similiar look to android. (As android uses linux kernel.)

Running your programs in a similiar manner to that of applications.
GNOME also gives you access to the SNAP store. Making software installation
as easy as windows. As well as giving access to tons of FOSS softwares

Link to GNOME website.

Difference between Ubuntu & Fedora

The main difference I'm going to point out is the package manager
Ubuntu uses the Debian Package managers, where Fedora uses RPM.

From my experience this mainly affects how certain proprietary softwares
are installed, as well as some differences or 'nuances' within Terminal.
